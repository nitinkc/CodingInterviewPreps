# 📚 Coding Interview Preparation

> Master algorithms and data structures through organized learning.
> **Not for beginners** — assumes comfort with [CodingBat](http://codingbat.com/java) style problems.

---

## 🚀 Quick Start

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

Visit: **http://localhost:8000**

---

## 🛠️ Other Commands

```bash
# Build static site
mkdocs build

# Build & test Java code
./gradlew build
./gradlew test
./gradlew clean   # clean build artifacts
```
