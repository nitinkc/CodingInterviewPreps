# Part 3: Advanced Algorithms & Graph Algorithms

This section covers graph theory, advanced algorithms, and optimization techniques critical for competitive programming and technical interviews.

## Topics Covered

### Graph Fundamentals
- **Graph Traversal**: BFS and DFS
- **BFS Theory**: Breadth-first search principles

### Shortest Path Algorithms
- **Dijkstra's Algorithm**: Single-source shortest path
- **Bellman-Ford Algorithm**: Negative edge weights support
- **Shortest Paths Applications**: Real-world problems

### Minimum Spanning Trees
- **Eager Prim's Algorithm**: Priority queue based MST
- **Kruskal's Algorithm**: Union-find based MST

### Strongly Connected Components
- **SCC Theory**: Understanding connectivity
- **SCC Algorithms**: Finding SCCs
- **Tarjan's Algorithm**: Linear-time SCC

### Advanced Topics
- **Bipartite Graphs**: Two-coloring and matching
- **Maximum Flow**: Network flow problems
- **Traveling Salesman Problem**: NP-hard optimization
- **Union-Find**: Disjoint set data structure
- **Arbitrage**: Graph-based financial algorithm

## Learning Path

1. Master **Graph Traversal** (BFS, DFS)
2. Learn **Shortest Path Algorithms**
3. Understand **Minimum Spanning Trees**
4. Study **Strongly Connected Components**
5. Explore **Advanced Applications**

## Key Concepts

- **Time Complexity**: O(V+E) for traversal, O((V+E) log V) for Dijkstra
- **Graph Representation**: Adjacency list, matrix, edge list
- **Weighted vs Unweighted**: Different algorithm choices
- **Directed vs Undirected**: Algorithm modifications needed
