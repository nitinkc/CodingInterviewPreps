# Part 2: Algorithms & Data Structures Part II

This section covers advanced string searching algorithms, compression techniques, and comprehensive coverage of sorting algorithms.

## Topics Covered

### String Searching
- **Ternary Search Trees** - Hybrid data structure for strings
- **Boyer-Moore** - Efficient string matching from right to left
- **Rabin-Karp** - Hashing-based pattern matching
- **Brute Force** - Naive string matching approach

### Data Compression
- **General Data Compression** - Overview of compression techniques
- **LZW Compression** - Lempel-Ziv-Welch algorithm

### Sorting Algorithms
- **Bubble Sort** - Simple O(n²) algorithm
- **Insertion Sort** - O(n²) algorithm, good for small arrays
- **Merge Sort** - O(n log n) divide-and-conquer
- **Quick Sort** - O(n log n) average case, fastest in practice
- **Shell Sort** - O(n log² n) gap-based sorting
- **Hybrid Sorting** - Combining multiple sort algorithms
- **Non-Comparison Sorting** - Counting, Radix, Bucket sorts

## Learning Path

1. Start with **Brute Force** for string basics
2. Learn **Boyer-Moore** and **Rabin-Karp** for optimization
3. Understand **Data Compression** concepts
4. Master **Sorting Algorithms** from simple to complex
5. Learn about **Hybrid** approaches for real-world applications

## Key Concepts

- **Pattern Matching**: Different string searching strategies
- **Rolling Hash**: Efficient pattern matching in Rabin-Karp
- **Bad Character Heuristic**: Skip characters in Boyer-Moore
- **Time Complexity**: Average, best, and worst cases
- **Space Complexity**: In-place vs extra space algorithms
- **Stability**: Preserving order of equal elements
