# Part 4: Algorithmic Java

This section covers advanced problem-solving techniques and algorithmic approaches.

## Topics Covered

### Fundamental Techniques
- **Selection**: Finding kth smallest/largest
- **Integral Problems**: Discrete integration
- **Tower of Hanoi**: Classic recursion problem

### Backtracking
- **N-Queens**: Constraint satisfaction
- **Knight's Tour**: Graph traversal
- **Maze Solving**: Pathfinding
- **Hamilton Path**: Graph problem
- **Sudoku**: Constraint problem

### Geometric Problems
- **Closest Pairs**: Divide and conquer

### Dynamic Programming
- **Theory**: Overview and techniques
- **Fibonacci**: Classic DP problem
- **Coin Change**: Unbounded knapsack variant
- **Knapsack**: 0/1 bounded knapsack
- **Bin Packing**: Resource allocation
- **Rod Cutting**: Optimal partitioning
- **Subset Sum**: Existence problem

### Graph Problems
- **Euler Paths**: Hamiltonian circuits variant

## Learning Path

1. Master **Selection** and **Integral Problems**
2. Learn **Tower of Hanoi** for recursion
3. Practice **Backtracking** problems
4. Study **Dynamic Programming** fundamentals
5. Solve **Geometric** problems
6. Explore **Graph** problems

## Key Techniques

- **Recursion**: Foundation for backtracking
- **Memoization**: Optimize exponential to polynomial
- **Constraint Propagation**: Backtracking optimization
- **Divide and Conquer**: Geometric problems
- **Graph Algorithms**: Traversal and optimization
